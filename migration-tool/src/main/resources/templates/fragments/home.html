• What is OIPA Migration Tool (OMT)
  • How to use OMT
  • OMT :

    Web-based centralized tool to migrate OIPA configuration code from one region to another.

  • Integrated authentication and authorization :

    Role based access control for each user for accessing different available features. E.g. Users with Developer privilege can only migrate from DEV to SANBOX region, QA onward migration can only be done by users with Release Manager privilege.

  • User friendly UI :

    Effective UI to perform each user their task quickly and effectively.

  • Move-sheet Upload :

    Predefined move-sheet can be uploaded for migrating OIPA configuration codes.

  • Migration Comment :

    Mandatory comments/justification for tracking rule migration.

  • User Friendly Dash-board :

    Informative dash-board for tracking rule migration for RM team. E.g. version, comments, date, migrated by, etc.

  • Rule Lock mechanism :

    OMT has Locking mechanism to prevent multiple users migrating same rule simultaneously.

  • Audit Tracking :

    OMT has feature to audit log and track each and every rules which has been migrated using the tool.

  • Reporting :

    OMT has feature to export migration summary in excel file.
